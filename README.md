# Quro

Quro is a lightweight **Quran audio listening app** designed to provide a simple and distraction-free experience for listening to Quran recitations and Islamic nasheeds.

The app is built as a **Progressive Web App (PWA)** so it can be installed on mobile devices and desktops and used like a native app.

This project is created with the intention of **Sadaqah Jariyah** — a continuous charity that benefits others.

---

## Features

• Quran Surah streaming  
• Favorite Surahs  
• Recently played history  
• Islamic nasheed streaming  
• Multiple Qari support (planned)  
• Installable Progressive Web App (PWA)  
• Lightweight and fast  
• No advertisements  
• Open source

---

## Why Quro?

General music platforms such as Spotify or YouTube mix Quran recitations with other content and distractions.

Quro focuses only on **beneficial Islamic audio** and provides a clean environment dedicated to listening to the Quran.

---

## Technology Stack

Quro is built using:

- **React**
- **Vite**
- **Progressive Web App (PWA)**
- **Cloudinary** (audio hosting)
- **LocalStorage** (favorites & recent history)

---

## Privacy

Quro does **not collect personal data**.

- No accounts
- No tracking
- No analytics
- No advertisements

Favorites and recent listening history are stored locally on the user's device.
